Comment entrer vos identifiants
Pour chaque plateforme, renseignez les identifiants demandés. Ils sont chiffrés et stockés uniquement sur votre Worker Cloudflare. Pour Facebook, il faut un token de page Meta — pas votre mot de passe personnel. Pour Telegram, créez un bot via @BotFather. Pour Discord, copiez le Webhook URL depuis les paramètres de votre salon.

Llama 3.3 70B — très rapide

Créez un compte sur platform.deepseek.com, allez dans API Keys → Create. Clé commençant par sk-
Obtenir la clé DeepSeek →
🌟
Mistral — généreux
Mistral Large — crédit offert
Créez un compte sur console.mistral.ai, allez dans API Keys → Create new key. Clé alphanumérique longue.
Obtenir la clé Mistral →

Images — GRATUIT Pollinations
Aucune clé requise — open source
Pollinations.ai est entièrement gratuit et sans clé API. Choisissez "Pollinations (gratuit)" dans Fournisseur d'images. Les images sont générées via leur service public.
Recommandation gratuite complète : Fournisseur texte = Groq (gratuit) + Fournisseur images = Pollinations (gratuit) → 0€/mois
